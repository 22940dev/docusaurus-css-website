Get the country details from ISO2 Code

GET /countries/[ciso]

Security#

This api use API KEY as an authentication method.

  • Name: X-CSCAPI-KEY
  • In: header

Request Parameters#

NameLocated inDescriptionRequiredType
cisoURLISO2 Code of CountryRequiredstring

Response#

CodeDescription
200Return a details of country
401Unauthorized.
404Not Found.
